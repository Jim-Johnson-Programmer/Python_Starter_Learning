## Step 1: Create/Open a folder for Lesson1

To open the folder, go to file menu and select "Open Folder..." Folder can be created in open folder dialog.

To add folders to your workspace, go to file menu and select "Add Folder to Workspace..." or click folder icon in the explorer sidebar.

## Step 2: Open the REPL

To open REPL command, go to Terminal menu and select "New Terminal". This will open a terminal at the bottom of VS Code window. Then at the terminal type py or python to start the Python REPL. You should see the REPL prompt `>>>`. You can now type Python commands at the prompt and press Enter to execute them one at a time. You can do math, assign variables, define functions, and more.

To exit the REPL, type `exit()` or `quit()` and press Enter, or press `Ctrl + Z` (Windows) or `Ctrl + D` (Mac/Linux) and press Enter.

## Step 4: Install the Python extension so that VS Code recognizes Python files and offers features like syntax highlighting, code completion, and debugging support.

1. Go to the Extensions view by clicking on the Extensions icon in the Activity Bar on the side of the window or by pressing `Ctrl + Shift + X`. The icon looks like four squares with one square rotated to look like a diamond.

2. In the search bar, type "Python" and look for the extension published by Microsoft.

3. Click on the "Install" button to install the extension.

4. Once installed, you may need to reload VS Code for the extension to take effect. You can do this by clicking the "Reload" button that appears after installation or by closing and reopening VS Code.

## Step 5: Create a Python file

There are two ways to create a Python file in VS Code.

1. File menu: Go to the File menu and select "New File". Type in the name "hello.py" that MUST end in .py and make sure to set the dropdown for file type to "Python" before saving.

2. Explorer sidebar: In the Explorer sidebar, click on the "New File" icon (a piece of paper with a plus sign) and type in the name "hello.py" that MUST end in .py.
