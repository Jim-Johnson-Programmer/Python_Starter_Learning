## Step 1: Create/Open a folder for Lesson1

To open the folder, go to file menu and select "Open Folder..." Folder can be created in open folder dialog.

To add folders to your workspace, go to file menu and select "Add Folder to Workspace..." or click folder icon in the explorer sidebar.

## Step 2: Open the REPL

To open REPL command, go to Terminal menu and select "New Terminal". This will open a terminal at the bottom of VS Code window. Then at the terminal type py or python to start the Python REPL. You should see the REPL prompt `>>>`. You can now type Python commands at the prompt and press Enter to execute them one at a time. You can do math, assign variables, define functions, and more.

To exit the REPL, type `exit()` or `quit()` and press Enter, or press `Ctrl + Z` (Windows) or `Ctrl + D` (Mac/Linux) and press Enter.
