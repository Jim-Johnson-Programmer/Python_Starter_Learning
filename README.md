# Python_Starter_Learning

This project shows the learning path for Python learning based on Dave Gray and freecodecamp.org.

## Tutorial Source

This learning path follows the comprehensive Python tutorial by Dave Gray on FreeCodeCamp:
- **Video Tutorial**: [Python for Beginners - Full Course](https://www.youtube.com/watch?v=qwAFL1597eM)

## About This Project

This repository contains code examples, exercises, and notes from following along with the tutorial. It's designed to help beginners learn Python programming through structured, hands-on practice.

## Credits

- [Dave Gray Teaches Code YouTube Channel](https://www.youtube.com/davegrayteachescode)

## Getting Started

To follow along with this learning path:
1. Watch the tutorial video linked above
2. Clone this repository
3. Follow the code examples and exercises
4. Practice and build your Python skills!
